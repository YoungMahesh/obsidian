### context

### commands
- ctrl+p = commands
	- move through select: n = next; p = previous
- escape: stop current request
- `/connect` - connect with provider
- `/models` - switch model
- `/new` - start new session
- `/sessions` - view history
	- delete-session: hover on session and press `ctrl+d`
