### set phrase
1. Open in i3wm: 
	- `autokey` to run in background
		- You will see it's icon in bottom-bar of i3 when it is running
		- To quit: right-click on icon -> click "Quit"
	1. `autokey-gtk` to run in background + open app window
2. Click New → Phrase
3. Once phrase and phrase-settings are set, then click 'Save' on top-bar

**phrase 1**
- Name: Arrow
- Phrase: `→`
- Phrase settings:
	- Paste Using: Clipboard (Ctrl+V)
	- Abbreviations: Set
		- Add: `->` (this is abbreviation) then Enter
		- Select: 
			- Trigger on: Space and Enter
			- Remove typed abbreviation
			- Ignore case of typed abbreviation
		- Click 'OK'
