Usually defined as: `type/subtype`

| MIME Type          | Meaning   |
| ------------------ | --------- |
| `text/plain`       | Text file |
| `text/html`        | HTML file |
| `application/pdf`  | PDF       |
| `image/png`        | PNG image |
| `application/json` | JSON      |
| `inode/directory`  | Folder    |
