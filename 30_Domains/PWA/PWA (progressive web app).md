
A **Progressive Web App (PWA)** is a **web application that behaves like a native app**.

### What makes an app a PWA:

To qualify as a PWA, your app typically includes:

1.  [[Service Worker]] (for offline + caching)
2. **Web App Manifest** (for installability, icons, theme)
3. **HTTPS**
4. Responsive UI
5. App-like experience (fast, reliable)

### What it provides:

* Installable on home screen
* Works offline or on poor networks
* App-like UI (no browser chrome)
* Background sync, push notifications
